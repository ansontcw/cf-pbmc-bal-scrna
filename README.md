# cf-pbmc-bal-scrna

This repository contains code used in Chapter 3 of Wong ATC's PhD thesis [1].
<br>Chapter title:
<br>**Integrated Blood-Airway Single-cell Transcriptomics Reveals Specific Myeloid and Lymphocyte Aberrations in Early-life Cystic Fibrosis Bronchiectasis**

## Overview
Computational analyses of single-cell RNA-sequencing (scRNA-seq) and multiplex immunoassay data from paired bronchoalveolar lavage (BAL) and peripheral blood of preschool-aged children with cystic fibrosis (CF) untreated with CFTR modulators, and age-matched non-CF controls.

## Data

**scRNA-seq:** Generated using the Cell Hashing with 10x Single Cell 3′ Reagent Kit v3.1 (Dual Index) Protocol. Demultiplexing and QC results can be explored at [this webpage](https://ansontcw.github.io/cf-pbmc-bal-scrna/).

**Multiplex immunoassay:** Generated using the Bio-Plex Pro Human Inflammation Panel (37-Plex) and Human Cytokine Screening Panel (48-Plex). For details, see Shanthikumar et al. [2].
